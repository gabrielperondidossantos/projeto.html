ESTRUTURA BÁSICA DE HTML

Para começar a fazer o seu primeiro site html vc precisa ter alguns comandos básicos entre eles são o html, head e bory.

todos eles tem um codigo de começo e um de fim que ele é defido por começo: < "codigo"> e fim: < / "codigo">

O primeiro que você deve usar é o html quando vc terminar de digitar o codigo de começo ele provavelmente ira fazer o de fim>

automaticamente, então primeiro digite o html de um enter e escreva o head, faça o mesmo e digite bory. com isso você fez

a estrutura básica de html, agora falta falar o que essas estruturas iram fazer. Primeiramente crie dentro de head o codigo

title no mesmo esquema que voçe escreveu os outros comandos, dentro dele escreva o titulo da aba do seu site.

Agora digite também dentro de bory a qual o titulo do site

Para você ver o resultado precisa salvar o que escreveu para isso aperte Ctrl S e após clique com o lado direito do mouse em cima

do arquivo no explore que você criou e clique em abrir comLive Server (open with Live Server), e ira abrir uma pagina no seu navegador

TAGS PARA FAZER SEU SITE

Texto em Negrito: Você pode usar <strong> para enfatizar uma palavra.

Texto em Itálico: Para usar itálico, utilize a tag <i>.

Texto Sublinhado: Use a tag <u> para sublinhar texto.

Texto Pequeno: A tag <small> diminui o tamanho do texto.

Texto com destaque: Use a tag <mark> para marcar texto.

LISTA

Você pode criar listas ordenadas e não ordenadas:

Lista Ordenada com a tag <ol>:

  1. Passo 1
  2. Passo 2
  3. Passo 3

Lista Não Ordenada com a tag <ul>:

  • Item A
  • Item B
  • Item C

PARA DIGITAR DENTRO DAS LISTAS ORDENADAS E NÃO ORDENADAS VC PRECISA USAR A TAG <li> DENTRO DELA QUE SE ESCREVE A LISTA.

OUTRAS TAGS IMPORTANTES

A tag HTML é a linguagem de marcação para a World Wide Web.

<sub>Cria um texto subscrito (abaixo da linha de base) EX: O símbolo químico da água é H<sub>2</sub>O.

<sup>Cria um texto sobrescrito (acima da linha de base) EX: A fórmula de Einstein é E=mc2.

<blockquote> Cita um bloco de texto, geralmente de outra fonte, EX: <blockquote>“A única coisa que eu sei é que nada sei.” - Sócrates<blockquote>

<del> Para cortar o texto EX: Este texto foi <del>cortado</del> da página.
